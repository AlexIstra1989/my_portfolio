About Me

I welcome you to my portfolio site. My name is Alexander. I live in Moscow. I deal with HTML-coding sites. I have a fully cross-browser, adaptive and valid layout. In order to meet the requirements of the web site layout I use Gulp project builder to automate the layout process, Sass preprocessor for faster site stylization and more flexible code, Pixel Perfect for pixel-to-pixel layout matching with provided layout, BEM and media queries for adaptive layout. The dynamics of the site is organized by our own jQuery code or customized plugins. At the end of the work the layout is checked in different browsers on different devices, the code is checked by the validator for compliance with standards and errors are eliminated.
